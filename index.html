DESCRIPTION

Introduction to Mobile Application Development, using Flutter” is a hands-on workshop which teaches the students to think about any software problem from a mobile developer standpoint and gives them the tools to develop a solution using Google’s latest mobile application

COURSE CONTENT
  1. Introduction to the data-flow on the internet
  2. Introduction to the simplicity of the dart language
  3. Widgets. Text, Containers, Columns, Rows, Lists
  4. Navigation and types of navigation
  5. State Management
  6. Making http requests and concluding with a sample application
Requirements
  1. Internet Connection
  2. Desktop/Laptop
WHAT YOU'LL LEARN

The students will have built the mindset to understand and breakdown how most modern day mobile applications work, and how uncomplicated the process of mobile development is
